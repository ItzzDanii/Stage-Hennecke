## 📌 Descrizione del Progetto

**Stage-Hennecke** è una piattaforma web sviluppata per raccogliere, organizzare e documentare tutte le attività giornaliere svolte durante il percorso di **PCTO (Alternanza Scuola-Lavoro)** presso l'azienda **Hennecke OMS S.p.A.**, leader mondiale negli impianti di produzione per il poliuretano.

L'obiettivo del sito è quello di fungere da diario di bordo digitale, rendendo tracciabili le competenze acquisite, i task completati e le tecnologie utilizzate giorno per giorno.

---

## 🛠️ Tech Stack (Tecnologie Utilizzate)

Ecco le tecnologie principali utilizzate per lo sviluppo del diario web:

| Frontend | Backend / Data | Strumenti |
| :--- | :--- | :--- |
| 🎨 **HTML5 & CSS3** (Layout & Stile) | 🗄️ **JSON / LocalStorage** (Gestione Dati) | 🚀 **Git & GitHub** (Version Control) |
| ⚡ **JavaScript (ES6+)** (Interattività) | 🌐 **Fetch API** (Caricamento contenuti) | 📝 **VS Code** (Editor di codice) |

---

## ✨ Funzionalità Chiave

*   📅 **Diario Giornaliero:** Visualizzazione cronologica delle attività svolte.
*   🏷️ **Tag e Categorie:** Filtro delle attività in base alla tipologia di lavoro (es. *Programmazione, Assistenza, Troubleshooting, Formazione*).
*   📱 **Design Responsive:** Interfaccia ottimizzata sia per PC che per smartphone.
*   📊 **Dashboard di Riepilogo:** Statistiche visive sulle ore totali accumulate e sulle macro-aree affrontate.
